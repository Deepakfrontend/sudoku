Sudoku Solver
==============
*Python Version 3.0+*

Naive Backtracking Algorithm
----------------------------

Run Command:

	python sudoku.py <input file name>
  example :- python sudoku.py <text_case_1.txt>

Smart Backtracking Algorithm (MRV)
----------------------------------

Run Command:

	python sudoku_MRV.py <input file name>
